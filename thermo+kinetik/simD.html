Hilfreiche Informationen

Konstanten:

  • Verdampfungsenthalpie von Wasser: ΔHvap = 43,9 kJ mol-1
  • Sublimationsenthalpie von Kohlenstoff: ΔHsub = 717 kJ mol-1
  • Gaskonstante: R = 8,314 J mol-1 K-1
  • Coulomb-Konstante: k = 8,99·109 N m2 C-2

Gleichungen:

  • Coulombsches Gesetz: F = k (q1q2)/r2
  • Gibbs-Energie: ΔGo = ΔHo - T ·ΔSo=-RT·ln(Keq)
  • Arrhenius-Gleichung: k = A · e - Ea/RT
  • Kinetische Energie und Geschwindigkeit: EK= ½ ·mv2

Simulation D: Chemisches Gleichgewicht

Hier geht es um das Erreichen von Gleichgewicht für zwei verschiedenen Arten chemischer Reaktionen.

Überblick der Simulationen:

  • Eine Animation (links) zeigt eine Darstellung jeder Reaktionen auf Partikelebene.
  • Ein begleitendes Diagram verfolgt die Konzentration der Partikel im Laufe der Zeit.
  • Sie können die Animationen jederzeit anhalten bzw. fortsetzen, indem Sie darauf tippen.
  • Wenn eine Animation gestoppt ist, können Sie auf Bereiche des Diagramms nebenan tippen, um genaue Werte abzulesen,

Beachte:

  • Jedes Teilchen in den Animationen kann als 1 μmol von Teilchen betrachtet werden.
